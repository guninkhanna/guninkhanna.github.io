Traffic Light Controller

• Programmed an FPGA using VHDL to act as a traffic light controller, involving finite state machines running on a clock
• Included binary and modulus counters for the clock divider which determines the flashing of the lights using the clock frequency
• Allows traffic to flow efficiently by changing lights based on car detection
• Technologies used: VHDL, Counters, FPGA
